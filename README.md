# COMMUNITY MANAGEMENT SYSTEM (Commu)

ระบบบริหารจัดการชุมชนสำหรับหมู่บ้าน “สร้างขุนศรี”  
**Tech Stack:** Django REST Framework + Vue 3 + Tailwind CSS

---

## ภาพรวมของระบบ (Overview)

**Commu** คือระบบจัดการข้อมูลชุมชนที่ช่วยให้คนในหมู่บ้านสามารถ:

- แจ้งรายงาน/ร้องเรียน → ผู้ใหญ่บ้านรับเรื่องได้ง่าย  
- ส่งคำขอความอนุเคราะห์  
- ขอคืนของ/สถานที่  
- ดูข่าวสารหมู่บ้าน  
- ดูข้อมูลผู้นำชุมชน  
- นัดหมายผู้ใหญ่บ้าน  
- ใช้แชทบอทถามข้อมูลหมู่บ้าน  
- ระบบแสดงสถานะออนไลน์ผู้ใหญ่บ้านแบบเรียลไทม์  

### ระบบรองรับ 3 บทบาท (Roles)

| บทบาท | สิทธิ์ |
|-------|--------|
| **Guest (ผู้เยี่ยมชม)** | ดูข้อมูลบางส่วน แต่ไม่สามารถเข้าถึงข้อมูลเชิงลึกได้ |
| **User (คนในหมู่บ้าน)** | ส่งรายงาน/คำขอ ดูข้อมูลเพิ่มเติม นัดหมายผู้ใหญ่บ้าน |
| **Admin (ผู้ใหญ่บ้าน/เจ้าหน้าที่)** | จัดการรายงาน อนุมัติคำขอ อัปเดตข้อมูลผู้นำ & ข่าวสาร |

---

## ฟีเจอร์หลักของระบบ (Core Features)

### 1. ข่าวสารหมู่บ้าน (Village News)
- แสดงข่าว/กิจกรรมล่าสุด  
- Admin เพิ่ม/แก้ไข/ลบข่าวได้  
- User กดดูรายละเอียดข่าวได้  

---

### 2. รายงานปัญหา (Issues & Reports)
- User ส่งรายงาน พร้อมหัวข้อ รายละเอียด เบอร์โทร และรูปภาพ  
- Admin สามารถ:
  - ดูรายละเอียดแบบ Popup  
  - รับเรื่อง / เปลี่ยนสถานะ  
  - บันทึกโน้ตทุกครั้งที่มีการอัปเดตสถานะ  
- ระบบเก็บ **ประวัติการเปลี่ยนสถานะทุกรอบ**  

---

### 3. คำขอความอนุเคราะห์ (Requests)
- User ส่งคำขอ เช่น ขอใช้อุปกรณ์หรือสถานที่  
- Admin อนุมัติ / ปฏิเสธ พร้อมเพิ่มโน้ตประกอบ  
- แสดงสถานะคำขอแบบ Real-Time  

---

### 4. ข้อมูลหมู่บ้าน & ผู้นำชุมชน
- แสดงข้อมูลหมู่บ้าน  
- แสดงโปรไฟล์ผู้นำชุมชน  
- Guest ดูได้เฉพาะข้อมูลพื้นฐาน  
- User ดูข้อมูลเพิ่มเติมได้  
- Admin มีเมนู “อัปเดตข้อมูล” สามารถเพิ่ม/แก้ไข/ลบโปรไฟล์  

---

### 5. ระบบนัดหมายผู้ใหญ่บ้าน (Appointment)
- User เลือกวัน/เวลานัดหมาย  
- Admin จัดการการนัดหมายทั้งหมด  
- มีระบบแสดงสถานะออนไลน์/ออฟไลน์ของผู้ใหญ่บ้านแบบ Real-Time  

---

### 6. ระบบแชทบอท (AI Chatbot)
- ถามข้อมูลทั่วไปเกี่ยวกับหมู่บ้าน  
- ระบบตอบอัตโนมัติด้วย AI  
- ใช้งานง่าย ผ่านหน้าเว็บ  

---

## สถาปัตยกรรมของระบบ (System Architecture)

### Backend
- Django 5 + Django REST Framework (DRF)  
- JWT Authentication  
- PostgreSQL  
- โมเดลหลัก:
  - User  
  - Report  
  - Request  
  - Appointment  
  - VillageInfo  
  - LeaderProfile  
  - News  

---

### Frontend
- Vue 3 + Vite  
- Tailwind CSS  
- Pinia (State Management)  
- Axios สำหรับเชื่อมต่อ API
